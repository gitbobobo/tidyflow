# 人类使用反馈

警告：阅读此文档请注意验证内容是否过时，请及时标记已处理的反馈项。

- [ ] 右侧面板自主进化的正在运行区域,点击查看聊天详情后，点击新会话，聊天历史清空，目前为止是正常的，但当收到新的流式回复时，又会继续显示回复内容。
- [ ] 右侧面板自主进化的正在运行区域,点击查看聊天详情后，如果连接的是 ACP 协议的编码工具，只会显示流式输出的新消息，没有加载最新的50条历史消息。
- [ ] 右侧面板自主进化的历史循环区域，点击查看交接文档，应该支持查看各个代理的交接文档，而非某个阶段的。

- [ ] 重新设计聊天消息相关的协议，确保由后端解析并转换后，前端可以直接用来渲染组件，无需添加自己的适配逻辑。

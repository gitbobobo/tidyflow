# Debug Panel Packaging Checklist

## Pre-Release Verification

1. [ ] Cmd+Shift+D opens Debug Panel
2. [ ] Panel shows Core status (state, port, pid)
3. [ ] Panel shows auto-restart count
4. [ ] Panel shows WebSocket connection state
5. [ ] Panel shows WebSocket URL
6. [ ] Refresh button loads log tail
7. [ ] Copy button copies log to clipboard
8. [ ] Reveal in Finder opens ~/Library/Logs/TidyFlow/
9. [ ] Escape key closes panel
10. [ ] Panel does not appear in menu bar (hidden feature)

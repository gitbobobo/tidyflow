# 人类使用反馈

警告：阅读此文档请注意验证内容是否过时，请及时删除过时内容。

- [ ] 右侧面板自主进化页面的正在运行区域的 AI 工具图标显示问题：设置中配置通用实现使用 copilot，但图标在 T 项目中显示为 Opencode 的图标，在 M 项目中显示为正确的 Copilot 图标。
- [ ] 右侧面板自主进化页面的正在运行区域的运行中动画，已经存在底部的加载条了，右上角的圆点显得有点多余。
- [ ] 右侧面板自主进化页面的正在运行区域运行到只能提交阶段时，缺少查看聊天内容的按钮，实际上应该把它看作普通的代理阶段，与其他代理拥有相同的布局。
- [ ] 右侧面板自主进化页面的历史循环卡片不要使用彩色背景，改成淡一点的普通卡片颜色。
- [ ] 聊天界面的工具卡片连续出现时，根据使用的AI工具的不同，有时完全贴在一起，有时互相直接距离又很远。后端解析数据时应该直接把消息列表平铺成一层缓存起来，前端也能有一致的显示效果。
- [ ] 自主进化系统现在会验证阶段产物的更新时间要大于开始时间，这个字段应该交由系统维护而非代理维护。

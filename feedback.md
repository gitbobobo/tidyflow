# 人类使用反馈

警告：阅读此文档请注意验证内容是否过时，请及时删除过时内容。

- [ ] 右侧面板自主进化的正在运行区域，耗时统计逻辑需要修复：通用实现->验证失败->重新通用实现的流程中，重新通用实现会统计从第一次开始的耗时而非本次会话的耗时
- [ ] 右侧面板自主进化的正在运行区域,点击查看聊天详情后，点击新会话，聊天历史清空，目前为止是正常的，但当收到新的流式回复时，又会继续显示回复内容。
- [ ] 右侧面板自主进化的历史循环区域，点击查看交接文档，应该支持查看各个代理的交接文档，而非某个阶段的。
- [ ] 右侧面板自主进化的重试按钮，目前没有发现可点击的时机，需要优化一下循环恢复机制。
- [ ] 右侧面板自主进化开始循环后，需要支持动态增加/减少轮次数量
- [ ] 聊天界面的终端卡片需要改为在标题区域显示终端命令
- [ ] ACP 协议的很多工具调用都没有正确显示对应的工具卡片，需要比较真实数据格式或借鉴成熟项目的处理逻辑
- [ ] 自主进化系统的各个实现代理的提示词，在验证失败后重实现时，应该专注于修复验证失败的问题，而非普通的提示词
